Apache is a web server. You can also run your web app using Apache web server as follows :
  1. Download the Apache web server from Apache HTTP Server (I am presuming you run Windows 7 or later version 64bit). Be sure that you have installed the latest C++ Redistributable Visual Studio 2015 before installing Apache.

  2. Extract the downloaded file using Winrar Or similar sofware.

  3. Open httpd.conf file residing inside conf folder in a Notepad or any Text editor. Better use Notepad++


  4. Edit line number 37 as shown ServerRoot "path-to-your-Apache24-folder". In my case it is : ServerRoot "F:/Server/Apache24"


    5. Note: "path-to-your-Apache24-folder" is the root-directory of your Apache Web Server.

  5. Edit line number 245 as DocumentRoot "path-to-your-web-app-directory". In my case it is : DocumentRoot "F:/Server/Apache24/htdocs". By default Apache looks for files to server in the htdocs directory.


    6. Note: By path-to-your-web-app-directory I mean the folder where you downloaded or cloned the github repository.

  6. Edit line number 246 as < Directory "path-to-your-web-app-directory" >.
    mine is < Directory "F:/Server/Apache24/htdocs" >

    7. Note: Web server by default look for some index.html file to host. So we keep a copy of this html file in the root of our to-be-served directory.


  7. Goto bin folder and from there run the ApacheMonitor.exe.


  8. Right-click on ApacheMonitor.exe which should be in system tray in the taskbar and open it. And click on Start.


  9. Open your Web Browser and enter url localhost:8080.

    10. Note: If you have already some server listening on port 8080 or you face some problem then try changing port in line 58 of httpd.conf file to some other port like 8081 etc.

And that's pretty much all you need to do.
